Develop
--------

Made with [Electron](http://electron.atom.io).

Init:
```bash
$ npm install
```

Run:
```bash
$ npm run start
```

Build all:
```bash
$ npm run build
```

Build macOS:
```bash
$ npm run build:osx
```

Build Windows:
```bash
$ npm run build:win32
```

Build Linux:
```bash
$ npm run build:linux
```
